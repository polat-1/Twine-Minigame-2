# Twine-Minigame-2
## Devlog
I gave the player the branching options I did this project because I wanted to create an escape room type adventure, similar to the henry stickman games or ms paint adventures. I utilized the looping mechanic to allow the player to restart if they died and to go forward and back through the various rooms. 

My thought process behind my game design was to make an eccentric and dramatic cat who convinced themselves they were an all-powerful legendary war lord, and have them struggle to escape a tiny apartment.

I also wanted to include visuals, so I drew a few, uploaded them to an image hosting site, and coded the links in.
